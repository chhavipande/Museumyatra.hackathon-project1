## About
Museum Tracker is a hackathon-built website that helps users discover museums and track the museums they have visited.  
The goal of the project is to encourage cultural exploration by giving users a simple way to record and reflect on their museum visits.

## Features
- Browse and discover museums
- Track and record visited museums
- Clean and user-friendly interface

## Tech Stack
- HTML
- CSS
- JavaScript
- Netlify (deployment)

## Planned Features
Due to time constraints during the hackathon, some planned features could not be implemented:
- AI-powered chatbot to assist users with museum recommendations and queries
- Personalized suggestions based on user interests and visit history
- Google Maps integration to display nearby museums and provide location-based discovery

## Future Improvements
- Integration of an AI chatbot and Google Maps for interactive guidance
- Enhanced filtering and search for museums
- Data visualization of user visit history
